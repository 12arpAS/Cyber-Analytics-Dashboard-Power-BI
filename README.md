# 🔐 Cyber Analytics Dashboard (Power BI)

## 📌 Overview

The **Cyber Analytics Dashboard** is an interactive data visualization project built using **Power BI** to analyze cybersecurity data. It helps identify threats, monitor network activity, and generate actionable insights through intuitive dashboards.

---

## 🎯 Objectives

* To analyze cybersecurity datasets effectively
* To detect anomalies and suspicious behavior
* To visualize security trends and patterns
* To support decision-making with real-time insights

---

## 🚀 Features

* 📊 Interactive and dynamic dashboards
* 🔍 Threat and anomaly detection insights
* 📈 Visual representation of cyber attacks
* ⚡ Real-time or near real-time data analysis
* 📌 User-friendly interface

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Microsoft Excel / CSV (Data Source)**
* **DAX (Data Analysis Expressions)**
* **Power Query (ETL Process)**
